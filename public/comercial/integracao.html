Integração Comercial

Este material foi criado para que qualquer pessoa, mesmo sem experiência em vendas, consiga entender como funciona o nosso processo e executar atendimentos com qualidade.

Nosso objetivo: não é vender mais caro. É vender o serviço correto para a necessidade real do cliente.

1. Mentalidade Comercial da Empresa

O QUE FAZEMOS
  • Entendemos a necessidade do cliente
  • Indicamos a melhor solução
  • Evitamos venda errada
  • Pensamos no longo prazo
O QUE NÃO FAZEMOS
  • Empurrar plano mais caro
  • Ignorar necessidade do cliente
  • Vender serviço errado
  • Queimar relacionamento por venda rápida
Venda errada gera problema, suporte, cancelamento e retrabalho. Venda correta gera fidelização.

2. Atendimento ao Cliente

Todo atendimento deve seguir uma lógica:

  1. Ouvir o cliente
  2. Entender a necessidade real
  3. Identificar o processo correto
  4. Executar o processo corretamente
Regra: Antes de fazer qualquer coisa no sistema, você precisa entender o que o cliente realmente precisa.

3. Visão Geral dos Processos

Novo Contrato

Cliente não existe na base

Reativação

Cliente já existia e quer voltar

Ponto Adicional

Outro local, mesmo cliente, mesmo plano

Upgrade/Downgrade

Mesmo local, mesmo cliente, mudança de plano

Mudança de Endereço

Mesmo cliente, novo endereço, com taxa em algumas filiais e assinatura quando houver renovação contratual

Mudança de Titularidade

Mesmo Endereço, novo titular

Escolher o processo errado é um dos maiores erros do setor.

4. Sistemas Utilizados

  • OPA! → Atendimento com o cliente
  • Inmap Sales → Leads e negociações
  • IXC → Cadastro, contrato, OS e agenda
Cada sistema tem uma função. Nunca pule etapas.

5. Agenda Técnica (IXC)

Antes de agendar qualquer instalação, é obrigatório verificar a agenda dos técnicos no IXC.

  • Verificar datas disponíveis
  • Confirmar com o cliente
  • Não prometer datas sem validar
Prometer data sem verificar agenda = erro grave.

6. Boas Práticas

  • Sempre confirmar dados com o cliente
  • Registrar tudo corretamente
  • Seguir o processo completo
  • Tirar dúvidas antes de errar
  • Revisar antes de finalizar
Dúvida não resolvida vira erro.

7. Erros Críticos

  • Escolher processo errado
  • Não verificar cliente no IXC
  • Erro no endereço
  • Erro no Wizard
  • Não verificar agenda técnica

8. Avaliação

Responda as perguntas abaixo com base no treinamento:

  1. Cliente quer internet em outro local. Qual processo usar?
  2. Cliente quer melhorar sinal na mesma casa. Qual processo usar?
  3. Cliente já existia e quer voltar. Qual processo?
  4. Qual informação é obrigatória para fazer viabilidade?
  5. Antes de agendar instalação, o que deve verificar?
  6. Se cliente quer plano diferente no ponto adicional, o que fazer?
  7. No ponto adicional, quantos produtos usar no padrão?
  8. Em quais situações a mudança de endereço passa a ter assinatura?
  9. Qual sistema é usado para falar com o cliente?
  10. Qual o maior erro em vendas?
O objetivo dessa avaliação é garantir que você entendeu o processo antes de atender clientes.
